## Getting Started

#### Installation

Install the dependencies:

```bash
npm install
```

#### Run with dev

1. Start the development server with HMR:

   ```bash
   npm run dev
   ```

2. Check lint and format code

   ```bash
   npm run format
   npm run lint
   ```

#### Building for Production

Create a production build:

```bash
npm run build
```
